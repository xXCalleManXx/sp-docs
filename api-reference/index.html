Simplified flow

Here is a very simplified flow, to give an idea of how the integration works.

sequenceDiagram User->>Superpay: User clicks on Superpay button in newsletter Superpay->>Store: Superpay requests to create a cart based on the customers information Store->>Superpay: Store returns a cart, with line items and other information Superpay->>User: Superpay shows the cart to the user
